Settings

The settings menu is in the upper right corner. It offers to switch between the graph and the circle packing layout. One can adjust some properties of these layouts and last there are options regarding the MSI image representation.

Layout Switcher

There are two buttons, Graph and Cicle Packing. A simple push changes to the visualizations that equals the label.

Layout Options (Graph)

To adjust the graph layout two properties can be changed, the number of iterations before the layout algorithm stops and the charge multiplier. The number of iterations describes the amount of steps the algorithm has stabilize. If the maximal number is reached and the layout is still not stable, it will stop and keep its current form.
The charge multiplier calculates the strength of repulsion of all nodes to each other. Setting it below/above one will decrease/increase the repulsion and the layout will contract/expand.

Layout Options (Circle Packing)

Circle Packing offers the possibility to resize all most inner circles depending on either the mean intensity of the corresponding MSI image or the degree of the corresponding graph node.

MSI Image Options

The Composition attribute affects the community images. In case of maximum every pixel of the community image consists of the maximum intensity of its member images. For average at each pixel the arithmetic mean of all member images is calculated.
The Colorscale property affects both, community image and m\z image. It will transform the value of each pixel linearly, depending on the chosen scale.

m/z Value List

The m\z value list consists of three parts. At the top the value of the node or circle is shown over which the mouse is hovering. Below that is lies the list with m/z values.
This list show all values of the dataset if the application is loaded and if the clear button is pushed or the root of the circle packing is clicked. With a click on a community node/circle or a m/z node/circle the list shows just the values of the corresponding community.
If a value in the list is selected, the associated m/z image is shown in the image area, the correpsonding circle becomes encircled and depending whether the node is expanded or not itself or the community node is encircled too.

Image Area

The image area shows two different kind of images triggered by selection of m/z values or nodes.
The upper image is called "community image". It presents an aggregated image of all images within the selected community node. The lower image, called m/z image, presents the intensity map of a chosen m/z value.
Selection the option creates an overlay image between the community image / m/z image and the three component pca image. The pca image is used as background. Intensity values in community image and the m/z image above a selected thresholds will set the alpha value to 1. This visualizes in which areas the community image and the m/z image overlap with the pca result.

Graph

A big node in the graph represents a community while a small node represents either a one member community or a single m/z value.
Communities can be expanded via CTRL + left click or command + left click (OS X). Using the same procedure on nodes in the expanded communities area will collapse them.
Left clicking a node will active its respective image in the image are, as well as limiting the m/z value list to values that are part of the selected community.
Holding left click allows for dragging and repositioning of nodes.

Circle Packing

The cicle packing view is an alternative representation of the hierarchical organisation of the graph structure. Again, big nodes are communities while small nodes are single m/z values. A left click activates the respective image in the image area.
